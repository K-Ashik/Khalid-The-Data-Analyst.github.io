Case Study 1: How does a bike-share navigate speedy success?
Data Analysis

This analysis is based on the Divvy case study “‘Sophisticated, Clear, and Polished’: Divvy and Data Visualization” written by Kevin Hartman (found here: https://artscience.blog/home/divvy-dataviz-case-study. The purpose of this script is to consolidate downloaded Divvy data into a single dataframe and then conduct simple analysis to help answer the key question: “In what ways do members and casual riders use Divvy bikes differently?” In this case study, I have performed data analysis for a fictional bike-share company in order to help them attract more riders. Along the way, I performed numerous real-world tasks of a junior data analyst by following the steps of the data analysis process: Ask, Prepare, Process, Analyze, Share, and Act.

Case Study 2: How can a wellness company play it smart?
Data Analysis

Bellabeat is a high-tech manufacturer of health-focused products for women. Their aim is to analyze the usage of their products in an effort to gain insight into how people are already using smart devices. In turn, this analysis should produce high-level recommendations to stakeholders with the aim of increasing their marketing strategies and becoming a larger player in the global smart device market. Some key questions that will be used to guide this analysis are : What are some trends in Fitbit smart device usage? How could these trends influence the Bellabeat marketing strategy?.
